Overview

This is the website for Art of the Web (DSGN 234). This is an interaction design class taught by Nika Simovich Fisher. All assignments, notes, readings, and other materials will be housed on this website.

The class meets on Mondays and Wednesdays from 2pm to 5pm. This class will be remote for the duration of the Spring 2021 semester.
